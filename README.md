battlestar-mathematica
======================

A math game created for the good folks over at [Belly](https://bellycard.com/).

### Using:
==========

- Node.js 0.10.25
- Express 3.4.8
- Socket.io 0.9.16
- Mocha 1.17.1
- Should.js 3.1.3
- Math.js 0.18.1
- RequireJS 2.1.11
- jQuery 2.1.0
- HTML5 Boilerplate 4.3.0
